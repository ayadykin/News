# News webserver

=====

An OTP application

## Installation instructions

1. Download dependencies `rebar get-deps`

2. compile `rebar compile`

## API

Supported methods:

*   `GET /`: read all news

*   `POST 

*   `PUT 

*   `DELETE /<news_id>`: delete news
